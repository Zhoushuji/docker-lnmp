# docker-lnmp

用 Docker 快速构建 Nginx + PHP + MySQL 开发/生产环境。

### 使用方法：

```
将项目克隆至本地
$ git clone https://github.com/getnas/docker-lnmp.git

进入项目目录，根据需要编辑相关配置文件。

启动项目
$ docker-compose up -d

```

浏览器访问 http://ip-address

可以根据需求修改 docker-compose.yml 配置文件。

### 进入容器

使用如下命令进入容器

```
docker exec -it container_name /bin/bash
```
